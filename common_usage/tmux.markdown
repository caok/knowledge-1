## some tmux usage


```ruby
tmux new -s demo

tmux attach -t demo

Ctrl + b  && d #exit

Ctrl + b && %  #split

```

more usage


```ruby
^b , <newname> # rename current session
^b w  #list windows
^b #  #goto windows #
^b l  #goto last window
^b n  #goto next window
^b p  #goto previous window
^b s  #list sessions

^b c  #create another shell/window


```
